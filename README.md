# モダン ToDo (Web + PHP + SQLite3)

インラインで編集できるモダンな Todo 管理 Web アプリケーション。  
タスク、見出し、水平線を自然に混在できる構造で、ドラッグ＆ドロップによる並び替えにも対応。

## ✨ 主な機能

### 📝 複数のアイテムタイプ
- **タスク**: チェックボックス + テキスト入力
- **見出し**: 太字・大きめフォント（h2相当）
- **水平線**: 薄いグレーの区切り線

### ⌨️ キーボードショートカット
- `/h` → 見出しに変換
- `/hr` → 水平線を挿入
- `Enter` → タスクを追加して次の入力へ
- `Ctrl+Delete` → タスクを削除

### 🎨 モダンな UI
- インラインで直接入力・編集
- ドラッグ＆ドロップで並び替え
- ホバーで表示される操作ボタン（≡ ドラッグ、× 削除）
- 完了タスクは削除線で表示
- #FAFAFA 背景で目に優しいデザイン
- Noto Sans JP フォントで美しい日本語表示

### 📱 モバイル対応
- タッチ操作に最適化
- レスポンシブデザイン

## 🚀 使い方

1. このフォルダでPHPビルトインサーバーを起動:
   ```bash
   php -S localhost:8000
   ```
2. ブラウザで http://localhost:8000 を開く

3. 基本操作:
   - 最下部の入力欄からタスクを追加
   - `/h` と入力して見出しを作成
   - `/hr` と入力して水平線を挿入
   - タスクをドラッグして順序を変更
   - チェックボックスで完了/未完了を切り替え
   - ホバー時に表示される `×` で削除

## 🪟 付箋風の小ウィンドウ化（Windows）
- Chrome/Edge: 右上メニュー → 「保存と共有」→「ショートカットを作成」→「ウィンドウとして開く」

## 📁 構成
- `index.html` : UI
- `style.css`  : スタイル（モダンデザイン）
- `script.js`  : フロントロジック（ドラッグ＆ドロップ、ショートカット対応）
- `api.php`    : CRUD + 並び替え API（SQLite3, PDOなし）
- `fonts/`     : Noto Sans JP フォント
- `todo.db`    : 初回アクセス時に自動生成（.gitignore 対象）

## 🗄️ データベース
既存のデータベースは自動的にマイグレーションされます。
- `type`: アイテムの種類（task/heading/divider）
- `sort_order`: 並び順
- `parent_id`: 将来のサブタスク対応用
